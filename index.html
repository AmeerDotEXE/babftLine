Create Line

White wall represents the team flag direction.

Usage example: place a block with two of it's corners touching the start and end point of your line, then get the scale of the block using the scale tool and input it down below, lastly select starting and ending points. you will get rotations needed for the line.
(make sure match rotation is off when placing the block for the line)
